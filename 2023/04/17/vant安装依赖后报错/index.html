vant安装依赖后报错

“Internal server error: Failed to resolve import… from “src\plugins\vant.ts”. Does the file exist”

在使用 vant 组件库时，初始化项目后，使用 npm install 安装依赖后启动项目就报一下错误

根据错误提示可以发现，是由于找不到文件产生的异常，并且它需要找的是 vant 的一些东西。经测试，发现在项目的 node_modules 文件夹下有一个叫 vant 的文件夹，将其中 es 文件夹中的所有样式引入就不再报错。在项目根目录找到 vite.config.ts 文件并配置一下内容。

配置完成后，项目就可以正常运行了。
